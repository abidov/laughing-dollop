# laughing-dollop
### To run this app you have to install Python.
Clone this repo and cd to folder

Create virtual environment with command:

```python3 -m venv env```

Activate:

```source env/bin/activate```

Install all required packages:

```pip3 install -r requirements.txt```

Run the app:

```python3 main.py```

To run tests:

```python3 tests.py```
